# Project Overview

## Objective
Build an automated ordering system for Disfruta that allows customers to easily submit their recurring orders via text message, while automating invoice creation and delivery list management.

## Core Functionality
- Automated sending of personalized order forms via SMS
- Personalized Fillout forms pre-loaded with customer's previous order
- Automatic order processing and QuickBooks invoice creation
- Daily delivery list generation in Google Sheets
- Order confirmation texts to customers
- Automatic forwarding of customer text replies to owners

## Key Integrations
- **Twilio**: SMS notifications and reply handling
- **Fillout.com**: Customer-facing order forms
- **QuickBooks Online**: Invoice creation
- **Google Sheets**: Data storage and delivery reports

## Current Status
- Google Sheets structure defined (5 tabs)
- Documentation framework created
- Core workflows documented
- Text reply forwarding added as core feature

## Phase 1 Scope
Automated order form distribution, processing, and basic notifications.
