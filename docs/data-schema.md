# Data Schema

This document outlines the structure of the five main Google Sheets tabs used in the Disfruta Automation system.

## Products Tab
- SKU
- Product Name
- Description
- Category
- Price
- Active
- Staff Pick
- Notes

## Recurring Orders Tab
- QuickBooks ID
- Customer Name
- SKU
- Product Name
- Default Quantity
- Frequency
- Day of Week
- Active
- Last Order Date
- Notes

## Clients Tab
- QuickBooks ID
- Customer Name
- Phone Number
- Email
- Frequency
- Last Order Date
- Active
- Notes

## Notes Tab
- Captures customer notes from orders (complaints, sample requests, delivery issues, special instructions)

## Delivery Reports Tab
- Generates daily delivery lists for the driver
